A web dashboard for tracking expenses with sorting, pagination, and detailed transaction views.

Tools: React 18, Tailwind CSS, Gemini.

Technical Hurdle: Learning under Pressure.

The Challenge: I had very little time to finish this project. While I am familiar with Angular, I had to build this entirely in React. Additionally, the AI recommended using Tailwind CSS, a utility-first styling framework I had never used before.

The Approach: Because of the tight deadline, I couldn't learn React from scratch. I used the LLM as a "concept translator," asking it to explain React Hooks (useState, useEffect) by comparing them to Angular concepts I already knew.

The Hurdle: Managing Tailwind's massive class strings was confusing at first. I prompted the AI to: "Explain Tailwind classes by comparing them to standard CSS properties" and "Help me structure this React component so it doesn't break when I pass data between files." This allowed me to deliver a working, styled UI in a fraction of the time.
